# prismagram

Instragram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [] Request Secret
- [] Confirm Secret (login)
- [] Like / Unlike a photo
- [] Comment on a photo
- [] Search by user
- [] Search by location
- [] See user profile
- [] Follow / Unfollow user
- [] See the full photo
- [] Edit my profile
- [] Upload a photo
- [] Edit the photo (Delete)
- [] See the Feed
